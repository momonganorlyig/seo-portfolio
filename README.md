# Orly Momongan — Local SEO Specialist Portfolio

A fully self-contained, multi-page portfolio website for **Orly Momongan**, Local SEO Specialist focused on home service businesses and SEO content writing.

## Live Site

Deployed via GitHub Pages: `https://[your-username].github.io/[repo-name]/`

## Features

- 7 pages: Home, About, Services, Case Studies, Testimonials, Blog, Contact
- All images embedded as base64 (no broken images, works fully offline)
- Pure static HTML/CSS/JS — zero dependencies, zero build step
- Fully mobile responsive
- Before/after local grid rank map comparisons
- Google Business Profile optimization showcase
- Google Search Console performance data
- Real article writing samples with featured images

## Deployment (GitHub Pages)

1. Create a new GitHub repository
2. Upload `index.html` to the root of the repository
3. Go to **Settings > Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save**
6. Your site will be live at `https://[your-username].github.io/[repo-name]/` within a few minutes

## Local Preview

Just open `index.html` in any browser — no server needed.
